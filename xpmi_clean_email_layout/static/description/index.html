XPMI Clean Email Layout

Modulo che semplifica le email di notifica Odoo con un flag aziendale. Il logo e il nome azienda restano visibili.

Funzionalita

  • Nasconde il blocco titolo documento (es. "Your Transfer" + numero).
  • Nasconde la riga contatti aziendali (telefono, email, sito).
  • Nasconde il footer "Powered by Odoo".
